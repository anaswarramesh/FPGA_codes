Button-Controlled LED Blinking on Tang Nano 9K

Overview

This project demonstrates button-controlled LED blinking using Verilog on the Tang Nano 9K FPGA board.

Behavior
Press BTN1 → LED0 blinks
Press BTN2 → LED1 blinks
Press BTN1 + BTN2 together → LED2 blinks

The design uses:

FPGA onboard 27 MHz clock
Counter-based clock division
Active-low buttons
